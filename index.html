NOTES FOR TESTING

Thanks for testing!

Submit feedback here!

Specific Feedback I'm looking for:

  • What device did you use? Mobile, PC/Desktop, Mac, etc (if mobile, which phone?)
  • If you used a Mac, did it work? (I've only tested on a windows machine)
  • Are controls intuitive?
  • Did you beat it?
  • Is it too hard, too easy, or a decent balance of challenging but beat-able?
  • Did you notice any lagging? If so, where? (Some visual elements are fairly heavy on the GPU)

Recording: Collect data that will be super helpful for balancing stats!

  • When you start the game, it will automatically start tracking metrics
  • Once you beat the game or get a Game Over, a file will automatically download with a ".json" extension
  • You can upload the file to the feedback form
  • To disable the recording - press the backtick key (`)
  • A little panel will open on the right (pictured below)
  • Click "Stop + Export"

Controls

🖥️ DESKTOP

  • WASD / ←↑↓→ – Move
  • / SPACE – Lasers
  • / Q / E – Barrel roll
  • Shift (Left or Right) – Speed boost
  • R (hold) – Singularity Bomb
  • P – Pause

📱 MOBILE

  • 🕹️ Joystick – Move
  • A – Lasers
  • B – Barrel roll
  • X – Speed boost
  • Y – Singularity Bomb
  • Pause Icon (top-left) - Pause
